# Mein Deutsch 1.0 Stable

Сімейний офлайн-PWA для вивчення німецької.

## Готовий контент

- Інна: 5 250 карток — радіологія, КТ/МРТ, пацієнти, документація, професійне B2.
- Рома: 5 250 карток — робота, Ausbildung, документи, житло, банк, страхування, авто.
- Назар: 5 250 карток — математика, природничі науки, історія, географія, письмові роботи й презентації.
- Разом: 15 750 карток.

## Встановлення

Завантажте всю папку на Netlify або GitHub Pages. Застосунок працює як PWA, кешує всі контентні пакети та після першого повного завантаження доступний офлайн.

На iPhone: Safari → «Поділитися» → «На початковий екран».

## Надійність

Прогрес зберігається за стабільними ID у localStorage. У налаштуваннях є експорт та імпорт резервної копії. Нові тематичні JSON-пакети можна додавати через `content/catalog.json` без зміни ядра.

## Перевірка

```bash
python tools/validate-content.py .
```
