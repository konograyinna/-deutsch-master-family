# Mein Deutsch — стан проєкту

- Поточна версія: **2.1.0**
- Завершено: профілі, уроки, картки, озвучення, тести, локальний прогрес, резервні копії.
- Завершено у 2.1.0: групування курсів за цілями; Führerschein та «Документи й життя в Німеччині» доступні Інні й Ромі; окремий прогрес кожного профілю; профіль Назара орієнтовано на 7 клас.
- Не змінено: дизайн, формат навчання, наявні ID уроків і збережений прогрес.
- Наступний етап: **2.2.0 — екран «Сьогодні»**.
